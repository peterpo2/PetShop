# Project Context - PetShop

## Business

`ЗОО БАКАЛИЯ` is a small neighborhood pet shop in Sofia, Bulgaria.

The shop sells food, treats, accessories, hygiene products, beds, houses, bowls, leashes, collars, harnesses, and other products for dogs, cats, and other pets.

The brand should feel:

- local
- warm
- trustworthy
- practical
- friendly
- professional without feeling corporate

## Website Goal

Create a one page landing website that quickly answers:

- what the shop offers
- where the shop is located
- how to call the shop
- why customers should choose this local shop
- what kind of care and personal advice customers can expect

Primary visitor context: mobile users opening the site from Google Maps, search, or a phone.

## Visual Identity

Source asset: `PetShopLogo.png`

Core colors:

- warm yellow: main brand color and accents
- saturated blue: headings, icons, buttons, graphic structure
- coral/red: CTA accents and small details
- graphite/dark gray: text and high-contrast sections
- white/cream: readability and balance

Use the logo as the first visual anchor in header and footer.

## Target Site Sections

1. Hero
2. About the shop
3. Product categories
4. Why choose ZOO BAKALIA
5. Gallery / atmosphere
6. Review
7. Location and contacts
8. Final CTA
9. Footer

## Technical Direction

No stack is implemented yet.

Preferred first implementation:

- static site
- mobile-first CSS
- minimal JavaScript
- optimized images
- deployable behind Nginx on Contabo VPS

Do not introduce a database, CMS, checkout, or account system without a new explicit request.
